REPL Host Page

It takes two to tango.

This page must run ClojureScript code to allow for a REPL socket connection. That script is loaded below. This makes this page a potential listener for ClojureScript REPL connections.

Then this page must be accessible via a true web server. That happens via lein ring server, or your server solution of choice. You'll need to have this page actually loaded into browser, not just accessible in theory.

Finally, the separate terminal window must be launched. That's where austin comes in.
