This page demonstrates basic usage of the React.js without any use of ClojureScript at all, let alone Om or Kioo. This is implemented as straight up JavaScript, HTML, and CSS.

The tutorial also uses a few other libraries, such as JQuery and Showdown, but they are only to add a bit of flair. React has no external dependencies out of the box.

See the main React tutorial for a step-by-step walkthrough of this code.

Note that this tutorial does not implement any kind of server-side storage whatsoever. A static JSON file is used as a very crude mock database, and it is reloaded every few seconds. Thus, posted comments appear right away (because of the optimisitic update code in the tutorial), but they vanish as soon as the JSON file is reloaded. This would be easy to fix, but I haven't bothered, my goal is to move on to Om instead of vanilla React ASAP.
