This page demonstrates basic usage of the React.js without any use of ClojureScript at all, let alone Om or Kioo. This is implemented as straight up JavaScript, HTML, and CSS.

The tutorial also uses a few other libraries, such as JQuery and Showdown, but they are only to add a bit of flair. React has no external dependencies out of the box.

See the main React tutorial for a step-by-step walkthrough of this code.
